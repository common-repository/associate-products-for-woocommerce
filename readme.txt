This Plugin shows associate(related) products using shortcode. The plugin will show related products using shortcode.

== Description ==

The plugin will show related products using shortcode. You can use the shortcode on a product single page in WooCommerce. It will show products based on user input value in the admin backend.

1. Display associated products for WooCommerce
2. Set the associated product's heading text
3. Set the column number for display associated products
4. Use a shortcode for display associated products [associate-product]
5. Display associated products on product single page based on current product categories


Kindly let us know your feedback or comments to add more features to this plugin.

== Installation ==

1. Log in to your WordPress admin panel and go to Plugins -> Add New
2. Type **Associate products for WooCommerce** in the search box and click on the search button.

OR

1. Download and save the **Associate products for WooCommerce** plugin to your hard disk.
2. Log in to your WordPress and go to the Add Plugins page.
3. Click the Upload Plugin button to upload the zip.
4. Click Install Now to install and activate the plugin.
